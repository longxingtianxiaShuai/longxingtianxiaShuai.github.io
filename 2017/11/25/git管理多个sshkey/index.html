git管理多个sshkey

#如果你有一个自己的git账号 又有一个公司的git账号 你可能会用的这一点

我们可以在~/.ssh目录下新建一个config文件配置一下，就可以解决问题. 下面用的xunwei 你可以替换为任何字符

    1. cd ~/.ssh
    1. ssh-keygen -t rsa -C “github邮箱”
    1. 不要一路回车 第一个回车后是让你填写一个名字 比如我这里用的是 xunwei 所以第一个回车后写的就是xunwei
    1. 添加私钥 ssh-add ~/.ssh/xunwei
    1. 创建config文件
  • 用touch config.txt 创建一个txt文本
  • 把后缀改为 config
  • 添加以下内容

    github

    Host github.com
    HostName github.com
    PreferredAuthentications publickey
    IdentityFile ~/.ssh/xunwei
    User gaoshuai
